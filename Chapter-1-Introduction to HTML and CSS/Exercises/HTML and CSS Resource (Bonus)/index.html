This is paragraph

<p> This is Paragraph </p>

anchor tag is used to insert links

w3schools

<a href="https://www.w3schools.com/"> w3schools </a>

strong tag is used to show text in bold or thick

        
This is strong

<strong> This is strong </strong>

small tag is used to show text in small in size

        
This is small text

<small> This is small text </small>

italic tag is used to show text in italic style

        
This is italic text

<i> This is italic text </i>

underline tag is used to underline the text

        
This is underlined text

<u> This is underlined text </u>

Color property

        
Sets the color of the text

        
{Color: red}

Background color property

        
Sets the color of the background

        
{background-color: yellow}

Font size property

        
Sets the font size of the text

        
{font-size: red}

Height property

        
Sets the height of the element

        
{height: 120px}

Width property

        
Sets the width of the element

        
{width: 100%}

Padding property

        
Sets the distance between the content and the border

        
{padding: 10px}

Border property

        
Sets the border of the element

        
{border: 5px solid black}

Margin property

        
Sets the distance between the current element and nearby element

        
{margin: 50px}

Text-align property

        
Sets the text-aligment

        
{text-align:center}

Display property

        
Sets how the element should be displayed

        
{display: inline}
